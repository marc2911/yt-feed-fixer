# YouTube Subscription Feed Cleaner

A Chrome extension that removes **YouTube Shorts** and the **“Most relevant”** sections from the YouTube subscriptions feed, restoring a clean chronological experience.

---

## Installation

1. Go to the **Releases** section of this repository  
2. Download the latest `.zip` file  
3. Unzip the folder on your computer  
4. Open Chrome and navigate to extensions or via URL `chrome://extensions/`
5. Enable **Developer Mode** (top right toggle)  
6. Click **Load unpacked**  
7. Select the extracted folder

Done.
